## This repository contains the solutions for the Data Science Test Tasks.
- First task is to train a Named Entity Recognition (NER) model for the identification of mountain names. 

- Second task is to perform image matching of Sentinel-2 images.

- Each folder in repository contains the solution for the corresponding task with the README.md file that describes how to run the code.

- Also there are a report pdf format  files in each repository that contains some explanations and potential improvements for the solutions.
